# 📋 O que o Sistema Faz

## 🎯 Resumo
Você coloca PDFs, faz perguntas, e o sistema responde baseado no conteúdo dos documentos.

## ✨ Principais Recursos

### 📄 Com PDFs
- Lê todos os PDFs da pasta `base/`
- F  arquivos ao mesmo tempo
- Extrai todo o texto automaticamente

### 🧠 Busca Inteligente
- Encontra as partes mais relevantes dos documentos
- Usa IA para entender o que você está perguntando
- Seleciona apenas o conteúdo mais útil

### 💬 Respostas
- Responde apenas com base nos seus documentos
- Usa GPT-4o-mini para gerar respostas claras
- Avisa quando não encontra a informação

### 🖥️ Interface
- Interface web simples no navegador
- Digite a pergunta e clique em "Buscar"
- Mostra o status do sistema

## 🔄 Como Funciona

1. **Preparação** (só uma vez):
   - Sistema lê todos os PDFs
   - Divide o texto em pedaços menores
   - Cria um "índice inteligente" para busca rápida

2. **Quando você pergunta**:
   - Sistema busca nos documentos
   - Encontra as partes mais relevantes
   - Envia para a IA gerar a resposta
   - Mostra a resposta na tela

## ⚠️ Limitações
- Só funciona com arquivos PDF
- Precisa de internet (usa OpenAI)
- Responde apenas sobre o que está nos documentos
- PDFs com imagens podem não funcionar bem

## 💡 Para que Serve
- Estudar livros e artigos
- Analisar contratos
- Consultar manuais técnicos
- Pesquisar em relatórios
- Qualquer situação onde você tem muitos documentos e quer encontrar informações específicas
